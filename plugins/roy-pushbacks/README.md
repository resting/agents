# roy-pushbacks

10 skills that break Claude out of yes-man mode.

> Source: [@sairahul1 on X](https://x.com/sairahul1/status/2056292143472202110)

Claude's default mode agrees with you. It elaborates on your ideas, validates your plans, and produces polished mediocrity. These skills break that mode — each one puts Claude into a specific adversarial or analytical role designed for a specific job.

## Skills

| Skill | What it does |
|-------|-------------|
| `/steelman` | Builds the strongest possible case for your idea before you pitch it |
| `/holefind` | Finds every flaw, gap, and weak assumption before someone else does |
| `/8020` | Extracts the 20% of any topic that produces 80% of the results |
| `/skeptic` | Stress tests an idea the way a sharp investor or editor would |
| `/disagree` | Argues against you intelligently — breaks yes-man mode entirely |
| `/unasked` | Finds the questions you didn't think to ask, then answers them |
| `/twolevels` | Explains any concept like you're 10, then like you're a PhD |
| `/contrarian` | Argues the opposite of conventional wisdom, seriously |
| `/compress` | Brutal clarity filter — every word earns its place or it's gone |
| `/memorize` | Converts anything you want to learn into something that actually sticks |

## When to use each

- **Before a pitch or decision** → `/steelman` then `/skeptic`
- **Before sending a draft** → `/holefind` then `/compress`
- **When you've made up your mind** → `/disagree`
- **Starting something new** → `/unasked`
- **Learning a new domain** → `/8020` then `/memorize`
- **Writing contrarian content** → `/contrarian`
- **Explaining to mixed audiences** → `/twolevels`

## The problem these solve

In default mode, Claude optimizes for responses that *feel* helpful. Feeling helpful and being useful are not the same thing. These skills force Claude into roles where agreement is not an option — which is where the actually useful output lives.
